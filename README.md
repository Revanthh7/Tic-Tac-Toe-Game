# ❌⭕ Tic Tac Toe Game in C

A simple **2-Player Tic Tac Toe** game developed in the **C Programming Language**. This project demonstrates the use of arrays, functions, loops, conditional statements, and game logic.

## ✨ Features

- 🎮 2-Player Gameplay
- ❌ Player 1 uses X
- ⭕ Player 2 uses O
- ✅ Win detection (Rows, Columns & Diagonals)
- 🤝 Draw detection
- 🚫 Prevents invalid moves
- 📋 Menu-free, easy-to-play console interface

## 🛠️ Technologies Used

- C Programming
- Arrays
- Functions
- Loops
- Conditional Statements
- GCC / Turbo C / Code::Blocks / VS Code

## 🚀 How to Run

1. Compile the program.
2. Run the executable.
3. Enter a position from **1 to 9**.
4. Players take turns until one player wins or the game ends in a draw.

## 📸 Board Layout

```
 1 | 2 | 3
---|---|---
 4 | 5 | 6
---|---|---
 7 | 8 | 9
```
